Upgrading components

If an upgrade to VxFlex Manager has added new required fields to components within the template from which a service was deployed, the Upgrade Components button is displayed. While this action is not mandatory, certain service or resource functions are not available until this upgrade has been completed.

Click the Upgrade Components button to launch the Upgrade Service Components window. Fields in this window vary depending on which components contain newly required settings. Complete all the displayed fields and click Save.
