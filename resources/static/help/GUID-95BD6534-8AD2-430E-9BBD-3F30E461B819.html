How to perform common tasks: Managing Flex family components

Once VxFlex Manager is configured, you can then use it to perform the ongoing tasks necessary to manage your VxRack FLEX.

The following table describes typical tasks for managing VxRack FLEX components and what steps to take in VxFlex Manager to initiate each.

If you want to...Do this in VxFlex Manager...
View network topologyClick Services. On the Services page, select a service. On the Details page, click the Port View tab.
Update inventory (nodes, switches, VxFlex OS Gateway, and vCenter cluster)Click Resources and then click the All Resources tab. Click the check box for the resource you want to update and then click Run Inventory. After running the inventory, click Update Inventory on the Services page for any service that requires the updated resource data.
Add an existing serviceClick Services and then click +Add Existing Service.
Perform node expansionClick Services. On the Services page, select a service. On the Details page, under Resource Actions, expand the Add Resources list and click Node. The procedure is the same for new services and existing services.
Remove a nodeClick Services. On the Services page, select a service. On the Details page, under Resource Actions, click Delete Resources.
Enter service modeClick Services. On the Services page, select a service. On the Service Details tab, select a component and click Enter Service Mode.
Exit service modeClick Services. On the Services page, select a service. On the Service Details tab, select a component and click Exit Service Mode.
