Deploying service on ESXi clusters

On the Templates page, select the template created for this use case and click Deploy Service.

ASM performs the following actions when you deploy this service:

  • Identifies the necessary servers from the FC server pool specified.

  • Boots the server when if it is turned on and verifies the FC connectivity.

  • Creates the storage volume and server objects that contain the WWPNs on Compellent or VNX Storage. ASM creates the storage volume and server objects with the names specified in the template.

  • If Configure SAN Switch parameter is set to true, ASM performs the following actions:

    • Identifies the fault domain of the Compellent or EMC Storage created on the Brocade switches.

    • Configures the Brocade switch by creating a zone for the server including the WWPN of the FC adapters and the Compellent or EMC Storage. The zones are added to the active zone set.

  • Maps the server object to the Compellent or EMC volume.

  • Installs ESXi, creates virtual networking based on the template, and creates and formats the VMFS data store for the attached Compellent or VNX volume and configures multipathing settings.
