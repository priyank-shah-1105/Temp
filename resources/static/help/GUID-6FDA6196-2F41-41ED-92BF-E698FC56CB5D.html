Connection settings matrix

Field name Description

Type Of Directory Service

Type of directory service (currently available, Microsoft Active Directory).

Name

Active Directory configuration name as added in VxFlex Manager. For example: mydomain

User Name

Active Directory account that has privileges to search for users. The account name must be entered using the User Principle Name format. For example: administrator@mydomain.com

Password

Active Directory password for the account in the User Name box.

Host

Active Directory FQDN host name or IP address. For example: 192.168.0.5

Port

Active Directory port. For example: 389

Protocol

Protocol type of Plain or SSL. For example: Plain.
