Settings

Only a user with the administrator role can configure certain settings.

Go to Settings to complete the following tasks:

  • Configure automatically scheduled and manual backup and restore jobs.

  • Create the credentials that VxFlex Manager uses to access node, switch, VMware vCenter, Element Manager, VxFlex OS Gateway, and OS resources.

  • Access the Getting Started page.

  • Access application logs.

  • Manage OS image and firmware repositories.

  • View and cancel jobs.

  • Define existing networks.

  • Manage VxFlex Manager users.

  • Perform management tasks related to NTP settings, proxy node settings, SSL certificates, alert connector, and licensing for the VxFlex Manager.

For more information about roles and permission, see About Roles
