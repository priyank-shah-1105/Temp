Attribute settings matrix

Field name Description

Base DN

Distinguished name (DN) where the users are searched by VxFlex Manager. It is the distinguished name of the starting point for directory searches.

For example: CN=Users,DC=mydomain,Dc=com

Filter

Filters that enable you to define search criteria. For example: objectClass=*

User Name Attribute

Active Directory record attribute that represents the user name attribute. This attribute is mapped to VxFlex Manager user name attribute. For example: sAMAccountName

First Name Attribute

Active Directory record attribute that represents first name. This attribute is mapped to VxFlex Manager first name attribute. For example: givenName

Last Name Attribute

Active Directory record attribute that represents the user last name. This attribute is mapped to VxFlex Manager last name attribute. For example: sn

Email Attribute

Active Directory record attribute that represents the user email. This attribute is mapped to VxFlex Manager email attribute. For example: mail
