Deploying a service

You cannot use a template that is in draft state to deploy a service. Publish the template before using it to deploy a service.

To deploy a service:

  1. On the menu bar, click Templates.

  2. On the Templates page, click Deploy New Service.

    The Deploy Service wizard starts.

  3. On the Deploy Service page, perform the following steps:

    1. From the Select Published Template list, select the template to deploy a service.

    2. Enter the Service Name (required) and Service Description (optional) that identifies the service.

    3. To specify the RCM version to use for compliance, select the version from the Release Certification Matrix (RCM) list or choose Use VxFlex Manager appliance default catalog.

      NOTE: Changing the firmware repository might update the firmware level on nodes for this service. Firmware on shared devices will still be maintained by the global default firmware repository.

    4. Indicate Who should have access to the service deployed from this template by selecting one of the following options:

      • Grant access to Only VxFlex Manager Administrators.
      • Grant access to VxFlex Manager Administrators and Specific Standard Users. Click Add User(s) to add one or more standard users to the list. Click Remove User(s) to remove users from the list.
      • Grant access to VxFlex Manager Administrators and All Standard Users.

  4. Click Next.

  5. On the Deployment Settings page, configure the required settings.

  6. To configure the VxFlex OS Settings, choose one of the following options for VxFlex OS MDM Virtual IP Source:

    • VxFlex Manager Selected IP instructs VxFlex Manager to select the virtual IP addresses.
    • User Entered IP allows you to specify the IP address manually for each VxFlex OS data network that is part of the node definition in the service template.

  7. To configure OS Settings, select an IP Source.

    To manually enter the IP address, select User Entered IP. From the IP Source list, select Manual Entry. Then enter the IP address in the Static IP Address box.

  8. To configure Hardware Settings, select the node source from the Node Source list.

    • If you select Node Pool, you can view all user-defined node pools and the global pool. Standard users can see only the pools for which they have permission. Select the Retry On Failure option to ensure that VxFlex Manager selects another node from the node pool for deployment if any node fails. Each node can be retried up to five times.
    • If you select Manual Entry, the Choose Node list appears. Select the node by its Service Tag for deployment from the list.

  9. Click Next

  10. On the Schedule Deployment page, select one of the following options:

    • Deploy Now—Select this option to deploy the service immediately.
    • Deploy Later—Select this option and enter the date and time to deploy the service.
