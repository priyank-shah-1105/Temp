Initial setup

The first time you log in to VxFlex Manager, you are prompted with an Initial Setup wizard, which prompts you to configure the basic settings that are required to start using VxFlex Manager.

Before you begin, have the following information available:

  • Local network share that contains the VxFlex Manager license
  • Time zone of the virtual appliance that hosts VxFlex Manager
  • IP address or hostname of at least one Network Time Protocol (NTP) node
  • IP address or hostname, port, and the credentials of your proxy node (optional)
  • Networks for VxFlex Manager to access in your environment (optional)
  • Network information to configure your appliance as a DCHP node
  • Host IP or FQDN of the OpenManage Enterprise application (optional)
  • Unique software ID for the VxRack FLEX(optional)
  • Serial number associated with the VxRack FLEX (optional)
  • Connector settings for Secure Remote Services (SRS) Gateway (optional)

To configure the basic settings:

  1. On the Welcome page, read the instructions and click Next.
  2. On the Licensing page, select a valid license, enter the Service Tag, and click Save and Continue.
  3. On the Time Zone and NTP Settings page, configure the time zone of the virtual appliance, add the NTP information, and then click Save and Continue.
  4. (Optional) On the Proxy Settings page, select the Use a proxy node check box, enter the configuration details, and then click Save and Continue.
  5. (Optional) To configure the VxFlex Manager as a DHCP or PXE node, on the DHCP Settings page, select the Enable DHCP/PXE node check box, enter the DHCP details, and then click Save and Continue.
  6. (Optional) To configure the alert connector, on the Alert Connector page, select the Enable Alert Connector check box. Enter the connector settings, the OpenManage Enterprise and device registration details, and then click Next.
    NOTE: You can configure the alert connector after the initial setup by going to Settings > Virtual Appliance Management > Alert Connector > Add.
  7. On the Summary page, verify the license, time zone, proxy node, and DHCP settings. Click Finish to complete the initial setup.

After you complete the initial setup, you can edit the NTP, proxy node, DHCP, alert connector, or license information, click Settings in the left pane, and then click Virtual Appliance Management.
