Viewing storage group details

  1. On the home page, click Resources.

    The Resources page is displayed.

  2. In the All Resources tab, click a storage group from the resources list to view the details.

    In the right pane, you can see the basic information about the storage group, such as System Status, Management IP, Storage Center Name, Group Members, Volumes, Replay Profile, Free Group Space.

  3. In the Details pane, click View Details.

    The Storage Group details page displays detailed information about storage group in the following tabs:
    • Summary
    • Volumes
    NOTE: In ASM 8.4 release, the detailed information can be viewed only for Dell Resources.
From the Storage Group Details page, you can view the recent alerts about the storage, and additionally:
  • For EqualLogic Storage, you can open the element manager GUI of Group Manager.
  • For Compellent Storage, you can open the element manager GUI of Storage Center.
