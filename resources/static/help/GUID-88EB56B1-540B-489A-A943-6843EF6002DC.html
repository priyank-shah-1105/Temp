Networks

VxFlex Manager manages general purpose LAN, hypervisor management, hypervisor migration, OS Installation, hardware management, VxFlex OS data and VxFlex OS management networks.

Ensure that the following network prerequisites are met:

  • VxFlex Manager can communicate with the out-of-band management network.

  • VxFlex Manager can communicate with the OS Installation network in which the appliance is deployed.

  • VxFlex Manager can communicate with the hypervisor management network.

  • The DHCP node is fully functional with appropriate PXE settings to PXE boot images from VxFlex Manager in your deployment network.
