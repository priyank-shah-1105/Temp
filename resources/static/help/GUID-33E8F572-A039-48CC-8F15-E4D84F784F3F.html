Uploading a node configuration profile

The Upload Node Configuration Profile feature enables you to upload a configuration XML file to VxFlex Manager. The settings uploaded from this file are applied to the hardware configuration of the target node at deployment time.

To upload a configuration XML file:

  1. Click Upload Node Configuration Profile.

  2. In the Upload Node Configuration Profile window, click Browse.

  3. Select the file you want to upload and click Open.

  4. Click Continue and then click Save.
