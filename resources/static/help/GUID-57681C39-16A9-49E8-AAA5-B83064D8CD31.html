Viewing a Release Certification Matrix (RCM) compliance report

Perform the following steps to view a Release Certification Matrix (RCM) compliance report:

  1. On the menu bar, click Resources.

  2. Select the resource for which you want to view the compliance report.

  3. In the right pane, click the link corresponding to the RCM Compliance option.

    The Release Certification Matrix (RCM) Compliance Report page is displayed.

  4. Select the Firmware Components option to view the details of the firmware available on the selected resource.

    For example, Switches.

  5. Select the Software Components option to view the software components in the compliance report or to update the software components for a resource. T he resource must be a part of the ESXi service. However, if the resource is in the In Use state, it redirects you to the Service page.

    To update the non-compliant resources, click Update Resources.

    NOTE: If there is no RCM attached to the resource, you cannot perform the firmware update or software update.
