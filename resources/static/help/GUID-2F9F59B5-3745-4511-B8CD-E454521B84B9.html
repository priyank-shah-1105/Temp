Viewing rack node details

From the Node Details page, you can:

  • Open the remote console of the nodeâ€™s Integrated Dell Remote Access Controller (iDRAC).
  • View recent activities performed on the node.

  1. On the menu bar, click Resources.

  2. On the All Resources tab, click a rack node from the resources list to view its details.

  3. In the Details pane, click View Details.

    The Rack Node Details page displays the detailed information about the node on the following tabs:
    • Summary
    • Port View
    • Network Interfaces
    • Firmware/Software Revisions
    • CPUs
    • Memory
    • Local Storage
      NOTE: You can select one of the following options from the Filter By drop-down list.
      • Logical Disks
      • Physical Disks
  • Contents
  • Search
