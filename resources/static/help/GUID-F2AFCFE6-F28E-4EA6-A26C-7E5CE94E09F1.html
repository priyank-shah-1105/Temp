Adding OS image repositories

To add an OS image repository:

  1. On the OS and RCM Repositories page, click OS Image Repositories tab, and then click Add.

  2. In the Add OS Image Repository dialog box, enter the following:

    1. In the Repository Name box, enter the name of the repository. The repository name must be unique and case insensitive. If you have two repositories with the same name (regardless of case), you will encounter deployment errors.
    2. In the Image Type box, enter the image type.

    3. In the Source Path and Filename box, enter the path of the OS Image file name in a file share.

      To enter the CIFS share, see the format used in the following example: \\host\lab\isos\filename.iso

      To enter the NFS share, see the format used in this following example: Host:/var/nfs/filename.iso

    4. If you are using the CIFS share, enter the User Name and Password to access the share.

  3. Click Add.
