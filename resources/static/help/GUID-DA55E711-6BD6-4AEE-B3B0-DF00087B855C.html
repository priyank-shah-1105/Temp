Adding components to a service

You can add one or more node, volume, or network components to a new service deployed by VxFlex Manager, or to an existing service that was not originally deployed by VxFlex Manager. You can add components to a service for which deployment was successful or to a failed service deployment. You do not need to go back to the original template from which the service was originally deployed.

Standard users can add components only to a service for which they have permission.

To add components to a service:

  1. On the menu bar, click Services.

  2. On the Services page, select a service and click View Details in the right pane.

  3. On the Service Details page, in the right pane, under Resource Actions, click one of the following components from the Add Resources list:

    • Node — Add one or more nodes to the service
    • Volume — Add one or more volumes to the service
    • Network — Add one or more networks to the service
