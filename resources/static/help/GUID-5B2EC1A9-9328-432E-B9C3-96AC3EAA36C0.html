Configuring the alert connector

When you configure the alert connector, it registers your device with Secure Remote Services (SRS) using a unique software ID. This enables critical and error alerting for node and VxFlex OS resources managed by VxFlex Manager.

Before you configure the alert connector, ensure:

  • Remote Access Controller Admin (RACADM) is enabled on the iDRACs of the discovered nodes.
  • VxFlex OS Gateway is discovered and configured.
  • Primary MDM in the VxFlex OS cluster is valid and up and running.
  • OpenManage Enterprise application discovered all the nodes.
  • OpenManage Enterprise application and the iDRAC SNMP community strings are identical and SNMP is set to Version 2.
  • SNMP alert filtering is set to the same level in the iDRACs and the OpenManage Enterprise application.
  • Secure Remote Services (SRS) Gateway is configured in the data center and connected to SRS.

  1. On the Alert Connector page, select the Enable Alert Connector check box.

  2. Complete the following steps in the Connector Settings section:

    • Enter a node address for the SRS Gateway in the SRS Gateway Host IP or FQDN box.
    • Enter a valid port number in the SRS Gateway Host Port box. The default port number is 9443.
    • Enter the required username in the User ID box.
    • Enter the required password in the Password or NT Token box.

  3. Complete the following steps in the Open Manage Enterprise (OME) Settings section:

    1. Enter a node address for the OpenManage Enterprise application in the Host IP or FQDN box.

    2. Enter the required username in the Username box.

    3. Enter the required password in the Password box.

    4. Select a valid alert level from the Alert Filter menu.

    5. Select a time from the Alert Polling Interval menu.

      The recommended time is 10 minutes. The minimum time is 5 minutes or greater.

  4. Complete the following steps in the Device Registration section:

    • Enter your unique software ID in the Enterprise License Management Systems (ELMS) Software Unique ID box. For information about how to obtain the ID, consult the License Authorization email that you received.
    • Enter the unique 8-digit number associated with your VxRack FLEX in the Solution Serial Number.

    For example, V1234567.

  5. Click Save.

    • If you want to view VxFlex OS alert messages, go to VxFlex OS: Documentation Library, select Download the documentation package and then select the Dell EMCVxFlex OS User Guide. The VxFlex OS Alerts in SNMP, VxFlex OS GUI, REST, and SRS topic summarizes the following alert message information:
      • Alert message in VxFlex OS GUI
      • Alert message in REST
      • Alert message in SNMP Trap
      • Alert code (for SRS)
      • Severity
      • Recommended action
