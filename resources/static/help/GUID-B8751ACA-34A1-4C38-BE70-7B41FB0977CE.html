Updating the VxFlex Manager virtual appliance

Whenever a new version of the virtual appliance is available, VxFlex Manager displays a banner at the top of all pages to notify users of the new release. The banner displays only for users with the Administrator role. It displays until a user with the Administrator role takes an action.

The new virtual appliance banner will show up only if you have registered with SRS.

You can update the virtual appliance in either of the following ways:

  • Update the appliance by using Secure Remote Services (recommended).
  • Update the appliance from a local repository path.

If you plan to use Secure Remote Services to update the appliance, you must first configure the alert connector.

To update the VxFlex Manager virtual appliance:

  1. On the banner, click View Details on the Actions menu.

    Alternatively, on the menu bar, click Settings, and then click Virtual Appliance Management.

  2. In the Appliance Upgrade Settings section , you can see the Current Virtual Appliance Version and check the Available Virtual Appliance Version field to see if a newer version of VxFlex Manager is available.

  3. To the right of the Appliance Upgrade Settings section, click Edit.

    1. To update to the latest version using Secure Remote Services, select Update Appliance from configured Dell EMC Secure Remote Services.

    2. To upgrade from a specific version on a local repository path, select Update Appliance from local repository path, and enter the path in the Repository Path box.

    3. Click Save.

    If you indicate that you want to update the virtual appliance using Secure Remote Services, the Repository Path field on the Virtual Appliance Management page shows Dell EMC Secure Remote Services (SRS). Otherwise, the field shows the network path entered in the Edit Appliance Upgrade Settings dialog.

  4. At the top of the Virtual Appliance Management page, click Update Virtual Appliance.

    1. After you click Update Virtual Appliance, a dialog box displays with a warning message indicating that this action logs off all current users and cancels any jobs in progress.

    2. Click Yes to update your appliance.

      The update process displays messages indicating the progress of the update. Once the update is complete, the system restarts and you are redirected to the login page.
