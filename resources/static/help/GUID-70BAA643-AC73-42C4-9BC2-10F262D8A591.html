Building a template overview

The template builder feature allows you to build a customized template by configuring both physical and virtual components. On the template builder page, you can set the component properties. For example, you can create a template that provisions only physical nodes with operating systems on them.

NOTE: A newly created or a cloned template appears in a draft state on the Template page and remains in the same state until published.

You can configure node and cluster components in a template.

The template builder page displays a graphical representation of the topology created within a particular template. From this page, you can:

  • Build and publish a template

  • Delete a template

  • Import a template

  • Deploy a service (this feature is available only on published templates)
