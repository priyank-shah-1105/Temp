Working with a template from a previous release

You can use a template from version 3.0.1. You do not need to recreate the template from scratch.

VxFlex Manager automatically converts a template from version 3.0.1 to the new model for the current release, so no upgrade is required by the user. However, the template created in version 3.0.1 will be marked as invalid. To allow the template to function properly, you need to update the settings for the VxFlex OS Cluster component.
