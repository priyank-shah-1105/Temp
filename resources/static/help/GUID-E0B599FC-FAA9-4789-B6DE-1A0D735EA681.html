Viewing VxFlex OS details

  1. On the menu bar, click Resources.

  2. On the All Resources tab, click a VxFlex OS Gateway resource from the list to view its details.

    The right pane displays basic information about the resource, such as the unused and spare space for VxFlex OS. It also shows the number of protection domains, volumes, VxFlex OS Data Clients (SDC), and VxFlex OS Data Servers (SDS).

  3. In the right pane, click View Details.

    The Details page displays detailed information about the VxFlex OS Gateway resource on the following tabs:
    • Performance: Displays read, write, and total I/O operations per second (IOPS) and read, write, and total bandwidth. In the IOPS and Bandwidth sections, you can choose to view graphs showing data for the last hour, day, week, month, or year. Current IOPS and bandwidth (KB/S) always display in the top-left corner. The data is automatically refreshed every five seconds. Performance data is available when VxFlex OS is deployed to a service.
    • Storage: Displays the protection domain, storage pools, volumes, and mapped SDCs.
    • Nodes: Displays the protection domain, type (SDS or SDC), name, and IP address for the SDS or SDC.
