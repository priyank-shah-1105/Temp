About roles

Every VxFlex Manager user account can be assigned to any one of the following roles:

  • Administrator — Users with the administrator role can view all pages and perform all operations in VxFlex Manager and grant permission to standard users to perform certain operations.
  • Standard — Users with the standard role can view certain pages and perform certain operations based on the permission granted by an administrator. Also, standard users can grant permission to other users to view and perform certain operations they own.
  • Read-only — Users with the read-only role can view all operations but are not allowed to perform any operations. When a user logs in as a read-only user, VxFlex Manager does not allow the user to perform any operations.
