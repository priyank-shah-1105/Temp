Welcome to VxFlex Manager

VxFlex Manager provides IT operations management for VxRack FLEX. It increases efficiency by reducing time-consuming manual tasks required to manage VxRack FLEX operations. Using VxFlex Manager, you can deploy and manage new and preexisting VxRack FLEX environments.

VxFlex Manager includes the following features:

  • A dashboard that provides system configuration details and communicates health status for VxRack FLEX infrastructure elements and services
  • Release Certification Matrix (RCM) compliance monitoring
  • RCM remediation for nodes and switches
  • Hardware monitoring and alerting through Secure Remote Services to Dell EMC Support
  • Aggregated logging for troubleshooting with the ability to send logs through Secure Remote Services to Dell EMC Support

Use VxFlex Manager to define and capture infrastructure requirements to support a specific business outcome. Once defined, the process can be easily repeated through automation.
