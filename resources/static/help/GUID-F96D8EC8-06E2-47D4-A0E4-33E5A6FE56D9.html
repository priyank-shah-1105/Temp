Network types

You can use VxFlex Manager to manage various network types.

  • General Purpose LAN—Used to access network resources for basic networking activities.
  • Hypervisor Management—Used to identify the management network for a hypervisor or operating system deployed on a node.
  • Hypervisor Migration—Used to manage the network that you want to use for live migration. Live migration allows you to move running virtual machines from one node of the failover cluster to different node in the same cluster.
  • OS Installation — Allows static or DHCP network for OS imaging on nodes.
  • Hardware Management—Used for out-of-band management of hardware infrastructure.
  • VxFlex OS Data—Used for data traffic between VxFlex OS Data Servers (SDSs) and VxFlex OS Data Clients (SDCs).
  • VxFlex OS Data (SDC Traffic Only)—Used for VxFlex OS Data Client traffic only.
  • VxFlex OS Data (SDS Traffic Only)—Used for VxFlex OS Data Server traffic only.
  • VxFlex OS Management—Used for VxFlex OS system management.
