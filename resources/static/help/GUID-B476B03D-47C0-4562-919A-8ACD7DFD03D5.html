Updating the repository path

By default, there is a repository path. To update the repository path, perform the following tasks:

  1. On the menu bar, click Settings and then click Virtual Appliance Management.
  2. Click Edit beside the Update Repository Path option.
  3. Update the path in the Update Repository Path box.
  4. Click Save to save the updated path.
  5. On theVirtual Appliance Management page, under Update Repository Path, you can view the updated information regarding Current Virtual Appliance Version, Available Virtual Appliance Version, Repository Path.
  6. To perform the update, click Update Virtual Appliance. A message displays confirming that you want to update the virtual appliance.
  7. Click Update Virtual Appliance.
  8. Click Yes. The system restarts.
  9. Once the update and restore is complete, click Click to log in and provide your credentials.
