Discovery and onboarding of rack nodes

VxFlex Manager supports rack node discovery and allows you to onboard nodes by configuring the initial management IP address and iDRAC credentials.

To perform initial discovery and configuration, verify that the management IP address is set on the rack node and that VxFlex Manager can access the IP address through the network. While configuring IP addresses on the rack node, verify that VxFlex Manager can access any final IP address in a range used for hardware management, to complete discovery of these nodes.

To discover and add rack nodes to the resource pool:

  1. On the menu bar, click Resources and then click Discover.

  2. On the Welcome page, click Next.

  3. On the Identify Resources page, click Add Resource Type.

  4. From the Resource Type drop-down list, select Node.

  5. In the IP Address Range field, enter the iDRAC IP address.

  6. Select the default credential from the Credentials list and click Next.

    The Initial Resource Configuration page displays the discovered node.

    NOTE: It might take few minutes to discover the node.

  7. The IP Address check box is selected by default when the node is discovered. If you do not want to configure the discovered node, clear the IP Address check box.

  8. To configure the discovered node, under Manage Credentials, select the credentials from the Credentials drop-down list.

    To create new credentials, click Create New.

  9. Under IP Addressing, do one of the following:

    • To use the existing IP address, click Use existing IP Address.
    • To change the existing IP address, click Assign static IP Address from this networkand then select Existing Hardware Management Network from the drop-down list. To create a new hardware management network, click New.

  10. Click Finish.

    The discovered rack node is added to the node pool.
