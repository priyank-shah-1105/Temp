Configuring time zone and NTP settings

On the Time Zone and NTP Settings page of the Initial Setup wizard, you can set the time zone and configure the network time protocol (NTP) nodes used for time synchronization.

Configuring NTP adjusts your VxFlex Manager system time. Your current user session ends if you adjust the time forward. The time syncs 5â€“10 minutes after this step. If this occurs, log in to VxFlex Manager again and continue with the setup process.

  1. From the Time Zone list, select the time zone in which VxRack FLEX operates.

  2. To synchronize the time with the NTP node, enter the IP address or fully qualified domain name (FQDN) of a Preferred NTP Node and Secondary NTP Node (optional).

  3. Click Save and Continue.

To change NTP node information after the initial setup is complete, click Setting > Virtual Appliance Management. On the Virtual Appliance Management page, click Edit in the Time Zone and NTP Settings section.
