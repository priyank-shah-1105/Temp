Creating a user

Perform this procedure to create a VxFlex Manager user and assign a role to that user.

  1. On the menu bar, click Settings and then click Users.

  2. On the Users page, click Create.

  3. Enter a unique User Name to identify the user account.

  4. Enter a Password that a user enters to access VxFlex Manager. Confirm the password.

    The password length must be between 8–32 characters and must include at least one number, one capital letter, one lowercase letter.

  5. Enter the user’s First Name and Last Name.

  6. From the Role drop-down list, select one of the following roles:

    • Administrator
    • Standard
    • Read only

  7. Enter the Email address and Phone number for contacting the user.

  8. Select Enable User to create the account with an Enabled status, or clear this option to create the account with a Disabled status.

  9. Click Save.
