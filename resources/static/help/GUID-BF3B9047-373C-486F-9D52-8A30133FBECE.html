Templates

A template is an object in VxFlex Manager that represents resource types and their required configuration and topology. A template specifies requirements for the deployment of a set of infrastructure resources through VxFlex Manager's built-in automation workflows.

On the Templates page, you can access the default sample templates or create templates that meet your specific requirements. For most environments, you can simply clone one of the sample templates provided with VxFlex Manager and make modifications as needed. Choose the sample template that is most appropriate for your environment.

VxFlex Manager allows you to create VxFlex OS templates that support hyper-converged service deployments, as well as service deployments that divide roles, where some nodes are used only for compute resources and other nodes only provide storage resources.

After creating a template, the template is saved in a draft state. You must publish the template before deploying it.

Users with standard permission can view and use only those templates for which they have been granted permission by the administrator.
