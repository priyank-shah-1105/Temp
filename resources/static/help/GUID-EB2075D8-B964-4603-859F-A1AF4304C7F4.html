Cluster

After selecting Cluster on the template builder page, perform the following steps:

  1. In the Cluster Component dialog box, from the Select a Component list, select VMWare cluster or VxFlex OS Cluster.

    For a compute only or hyper-converged template, you need to include a VMWare cluster and a VxFlex OS Cluster component.

    For a storage only template, you only need to include a VxFlex OS Cluster component.

  2. Under Related Components, select the components that you want to map to the selected cluster instance.

  3. Click Continue.

  4. Under Cluster Settings, specify the settings that you want to configure on the cluster components and click Add.

    For more information about the cluster settings, see Cluster Component Settings
