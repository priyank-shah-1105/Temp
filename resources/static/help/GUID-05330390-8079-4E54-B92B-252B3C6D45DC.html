Deleting a network

You should not delete a network that is referenced in a template. This affects the services that are deployed using this template.

  1. On the menu bar, click Settings and then click Networks.

    The Networks page is displayed.

  2. Click the network that you want to delete, and then click Delete.

  3. Click OK when the confirmation message is displayed.
