Exporting a template

To export a template:

  1. On the menu bar, click Templates.

  2. Select the template that you want to export.

  3. In the right pane, click Export Template.

  4. In the Export Template to ZIP File window, enter values as follows:

    1. Enter a name for the template file in the File Name.

    2. If you have set an encryption password, select Use Encryption Password from Backup Setting to use that password. To set an encryption password, deselect this option.

    3. If you deselect Use Encryption Password from Backup Setting, two additional fields display. Enter a new password in the Set File Encryption Password box and enter the password again to confirm it.

  5. Click Export to download the file. Select a location to save the file and click OK.
