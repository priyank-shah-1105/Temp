Adding an attachment to a template

You can attach one or more files to a template by using the Add Attachment feature. The file size of each attachment cannot exceed 50 MB.

To add an attachment to a template:

  1. On the menu bar, click Templates. Select the template to which you want to add an attachment.

  2. In the right pane, click Add Attachment.

  3. In the Add Attachment dialog box, click Choose File to browse to the file you want to attach. Locate the file and click Open to attach the file.

    If you try to attach a file with the same name of already attached file, you get an error message. You can attach multiple files with different file names.

  4. If you want to view the file, select the attached file and right-click. From the Open with list, select Notepad. Click OK to view the file.

  5. To delete an attachment, click the delete icon. A confirmation message appears. Click Yes to confirm deletion.
