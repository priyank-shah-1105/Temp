Cloning a template

The Clone feature allows you to copy an existing template into a new template. A cloned template contains the components that existed in the original template. You can edit it to add additional components or modify the cloned components. To clone an existing template:

  1. On the menu bar, click Templates.

  2. Select a template and then click Clone in the right pane.

  3. In the Clone Template dialog box, enter a template name in the Template Name box.

  4. Select a template category from the Template Category list. To create a template category, select Create New Category

  5. In the Template Description box, enter a description for the template.

  6. To update the firmware and software while deploying a service using this template, select the Perform compliance check with selected RCM version check box and select an RCM version from the list.

  7. To grant access to standard users to use this template, click the Manage Service Permissions check box and then select any one of the following options:

    • All Standard Users — Select this option to provide access to all standard users.
    • Specific Standard Users — Select this option to provide access to specific users. Click + Add User(s) to add the users. To remove users added to list, select the user and click Remove User(s).

  8. Click Next.

  9. On the Additional Settings page, under Network Settings, select a network from the Select New Network list.

  10. Under OS Settings, do the following:

    1. Enter the OS administrator password in the OS Administrator Password box.

    2. Select the new OS repository from the Select New OS Repository list.

  11. Under Node Pool Settings, select the new node pool from the Select New Node Pool list.

  12. Click Finish.
