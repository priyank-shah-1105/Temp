Learn more about VxFlex Manager appliance storage usage

Whenever the storage usage for the VxFlex Manager appliance is 75 percent or higher, VxFlex Manager displays a banner at the top of the Dashboard to notify you of the current storage usage. The banner reports the current storage usage of the partition with the least amount of space available (for example, /var, /tmp, /lib, or /home). The banner reports the current space usage as a percentage of the available space. To remedy the situation, remove any unnecessary files in the partition.

If the storage usage is between 75 percent and 95 percent, the banner shows a yellow status, which indicates a warning. Above 95 percent, the banner shows a red status to alert you to a critical storage situation.

The Appliance Overview on the Dashboard also displays a gauge meter that shows the storage used by the appliance. The meter shows a green status as long as the storage usage is below 75 percent. Between 75 percent and 95 percent, the meter shows a yellow status. Above 95 percent, the meter shows a red status to alert you to a critical storage situation. To avoid encountering a critical situation, check the gauge meter periodically to be sure that the status remains green.
