Adding a template

The Add a Template feature allows you to create a new template, clone the components of an existing template into a new template, or import a pre-existing template.

For most environments, you can simply clone one of the sample templates provided with VxFlex Manager and make modifications as needed. Choose the sample template that is most appropriate for your environment.

To add a template:

  1. On the menu bar, click Templates.

  2. On the Templates page, click Add a Template.

  3. In the Add a Template dialog box, select one of the following options:

    • Clone an existing VxFlex Manager template
    • Upload External Template
    • Create a new template

    If you select Clone an existing VxFlex Manager template, select the Category and the Template to be Cloned. The components of the selected template are included in the new template.

  4. Enter a Template Name.

  5. From the Template Category list, select a template category. To create a category, select Create New Category from the list.

  6. Enter a Template Description (optional).

  7. To specify the RCM version to use for compliance, select the Perform compliance check with selected RCM version check box and choose the version from the RCM version list.

    NOTE: Changing the RCM version might update the firmware level on nodes for this service. Firmware on shared devices will still be maintained by the global default firmware repository.

  8. Specify the service permissions for this template under Who should have access to the service deployed from this template? by performing one of the following actions:

    • To restrict access to administrators, select Only VxFlex Manager Administrators.
    • To grant access to administrators and specific standard users, select VxFlex Manager Administrators and Specific Standard Users and perform the following steps:

      1. Click Add User(s) to add a standard user to the displayed list.

        To remove a standard user from the list, select the user and click Remove User(s).

      2. After adding the standard users, select or clear the check box next to the standard users to grant or block access to use this template.
    • To grant access to administrators and all standard users, select VxFlex Manager Administrators and All Standard Users.

  9. Click Next to select the remaining options to complete the wizard.
