Editing an OS image repository

To edit an OS image repository:

  1. On the OS and RCM Repositories page, click OS Image Repositories tab.

  2. From the Available Actions drop-down list, click Edit for a repository in an Error state.

    The Edit OS Image Repository page is displayed.

  3. Edit the Source Path and Filename and enter the user credentials.

    NOTE: You cannot modify the Repository Name and Image Type.

  4. Click Save.
