How to perform common tasks: Deploying a Flex family system

The following table describes typical tasks for deploying and provisioning VxRack FLEX and what steps to take in VxFlex Manager to initiate each.

If you want to...Do this in VxFlex Manager...
Define service and infrastructure requirementsClick Templates and then click Add a Template.
Deploy a new serviceClick Services. On the Services page, click Deploy New Service.

You can only deploy a service using a published template.
