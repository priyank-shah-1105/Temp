Editing an IP verification port number

You must verify what IP addresses are in use before they are deployed. VxFlex Manager uses default ports 22, 80, and 135 for verification.

  1. On the menu bar, click Settings and then click Virtual Appliance Management.

  2. Under IP Verification Port Number, click Edit.

  3. In the Port Numbers Used for IP Verification box, add or remove the port numbers used by VxFlex Manager to verify the IP addresses.

    Port numbers must be entered in comma-separated list, and must be between 1 and 65535.

  4. Click Save.
