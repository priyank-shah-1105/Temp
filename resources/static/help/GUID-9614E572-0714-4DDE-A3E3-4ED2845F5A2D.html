What's new in this release

VxFlex Manager release 3.1 introduces the following new features:

  • Support for the two-layer architecture
  • Extended support for existing deployments
  • Ability to add or remove volumes within a service
  • Extended support for Secure Remote Services (SRS) alerting on VxFlex OS nodes
  • Enhanced logging
  • Miscellaneous enhancements to existing features

Note that VxFlex Manager now refers to node(s) instead of server(s).
