Configuring proxy settings

If your environment uses a proxy node to communicate with external services, you must configure the proxy settings in VxFlex Manager.

To enable communication using a proxy node:

  1. On the Proxy Settings page of the Initial Setup wizard, select the Use HTTP Proxy Settings check box.

  2. In the Node Address box, enter the IP address or host name for the proxy node.

  3. In the Port box, enter the port number for the proxy node.

  4. If the proxy node requires credentials to log in, select the Use proxy credentials check box, enter the User Name and Password, and then reenter the password to confirm.

  5. To test the connection to the proxy node, click Test Proxy Connection.

  6. Click Save and Continue.

After the initial setup is complete, click Settings > Virtual Appliance Management to change the proxy settings. On the Virtual Appliance Management page, click Edit in the Proxy Settings section.
