Virtual appliance management

Virtual Appliance Management enables you to:

  • Update the VxFlex Manager virtual appliance
  • Generate a troubleshooting bundle
  • Edit time zone and NTP settings
  • Update repository path
  • Edit proxy node settings
  • Edit DHCP settings
  • Configure the alert connector
  • Edit the alert connector settings
  • Generate and download a certificate signing request (CSR) and upload the resulting SSL certificate
  • Upload a VxFlex Manager license

Updating the virtual appliance

If a new version of the virtual appliance becomes available, VxFlex Manager displays a notification banner at the top of the screen with the text A new version of the VxFlex Manager virtual appliance is available for download and install. View Details. To the far right of the banner, you should see an Actions menu that provides the following choices:

Select this option...To...
View detailsView details about the new release and download it.
HideHide the banner for the new release. This action applies to only the current user and session. If the current user logs off, the banner reappears when this or any other user logs on again. The notification banner also displays if another release becomes available.
Dismiss 30 daysDismiss the banner for this particular release for 30 days. This action applies only to the current user. The banner appears if a different user logs on. The notification banner also displays if another release becomes available.

The new virtual appliance banner will show up only if you have registered with SRS.
