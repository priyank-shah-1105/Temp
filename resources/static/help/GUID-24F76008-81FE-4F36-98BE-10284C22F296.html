Editing template information

To edit information for a template:

  1. On the menu bar, click Templates.

  2. On the Templates page, click the template that you want to edit and click Edit in the right pane.

  3. On the template builder page, In the right pane, click Edit. Update fields as needed.

    • In the Template Name box, modify the template name as needed.
    • From the Template Category list, select a template category. To create a category, select Create New Category from the list.
    • In the Template Description box, enter the description for the template.
    • To specify the RCM version to use for compliance, select the Perform compliance check with selected RCM version check box and choose the version from the RCM version list.
    • Specify the service permissions for this template under Who should have access to the service deployed from this template?. You can restrict access to administrators only, grant access to administrators and specific standard users, or grant access to administrators and all standard users.

  4. To specify the RCM version to use for compliance, select the Perform compliance check with selected RCM version check box and choose the version from the RCM version list.

    NOTE: Changing the RCM version might update the firmware level on nodes for this service. Firmware on shared devices will still be maintained by the global default firmware repository.

  5. Click Save.
