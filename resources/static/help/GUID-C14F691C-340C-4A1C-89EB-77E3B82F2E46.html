Uploading an external template

To upload a template:

  1. On the menu bar, click Templates.

  2. Click Upload External Template to upload the exported template.

    The Upload External Template page appears.

    For more information on exporting template, see Exporting Template.

  3. Click Browse to select the exported file from your directory.

  4. Select the Use Encryption Password from Backup Settings check box, if you have set the encryption password in Backup and Restore.

    Clear the Use Encryption Password from Backup Settings check box if you have not set the encryption password in Backup and Restore.

    If you clear the Use Encryption Password from Backup Settings, you must enter the encryption password in the Encryption Password field.

  5. Enter the template name in the Template Name box.

  6. Select a template category from the Template Category list.

    Select Create New Category if you want to create a template category.

  7. In the Template Description box, enter a description for the template.

  8. To specify the RCM version to use for compliance, select the version from the Release Certification Matrix (RCM) list or choose Use VxFlex Manager appliance default catalog.

    NOTE: Changing the firmware repository might update the firmware level on nodes for this service. Firmware on shared devices is maintained by the global default firmware repository.

  9. To grant access to standard users to use this template, select the Manage Service Permissions check box and click any one of the following options:

    • All Standard Users — Select this option to provide access to all standard users.
    • Specific Standard Users — Select this option to provide access to specific users. Click + Add User(s) to add the users. To remove users added to list, select the user and click Remove User(s).

  10. Click Upload and Continue.

    The Additional Settings page appears.

  11. Under Network Settings, select new network from the Select New Network list.

  12. Under OS Settings, do the following:

    1. Enter the OS administrator password in the OS Administrator Password box.

    2. Select the new OS repository from the Select New OS Repository list.

  13. Under Node Pool Settings, select the new node pool from the Select New Node Pool list.

  14. Click Finish.
