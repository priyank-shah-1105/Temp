Editing backup settings and details

You can change the location where backup files are saved or password required to access a back up file.

  1. On the menu bar, click Settings and then click Backup and Restore.

  2. On the Backup and Restore page, under Settings and Details section, click Edit. The Settings And Details page displays.

    Optionally, to indicate the network share location where the backup file is saved, enter a backup directory path in the Backup Directory Path box. Use one of the following formats:

    • NFS — host:/share/
    • CIFS — \\host\share\

    If the username and password are required to access the network share, enter a username and password in the Backup Directory User Name and Backup Directory Password boxes.

  3. To open the backup file, enter a password in the Encryption Password box.

  4. To verify the encryption password, enter the password in the Confirm Encryption Password box. The password can include any alphanumeric characters such as!@#$%*.

  5. Click Save.
