Understanding node pools

In VxFlex Manager, a node pool is a set of nodes grouped for specific use cases such as business units or workload purposes. An administrator can specify a set of users who can access these node pools.

Standard users can view details only for node pools for which they have permission.

From the Node Pools tab, you can view existing node pools.

Users with an administrator role can create, edit, or delete node pools.

Click a node pool from the list to view detailed information on the following tabs:

  • Nodes: Displays the number of nodes associated with the node pool.

  • Users: Displays the number of users with access rights to the node pool.
