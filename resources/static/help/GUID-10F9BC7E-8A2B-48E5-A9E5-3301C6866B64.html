Viewing template details

To view more details about a template:

  1. On the Templates page, select a template.

  2. In the right pane, click View Details.

    The topology of the components in the template appear in the template builder.

  3. To view all the component settings, click View All Settings in the right pane.

    The Template Settings page lists the details about the component configured in the template. For more information about component settings, see Component Types.
