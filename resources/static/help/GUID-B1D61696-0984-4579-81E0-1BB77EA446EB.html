Viewing storage details

The following are the steps to view the storage details:

  1. On the home page, click Resources.

    The Resources page is displayed.

  2. In the All Resources tab, from the Resource Type drop-down menu, select Storage.

    The list of storage array is displayed.

  3. Select a storage array to view the details.

    In the right pane, the following information about the storage is displayed:
    Filed Name Description
    Service Indicates whether the service is In Use state or Not in Use state.
    Firmware Status

    Indicates the firmware and software compliance state of the resources. For more information about the compliance status, see the RCM compliance status

    System StatusIndicates the system status.
    VolumesIndicates the number of volumes created.
    Replay ProfileIndicates the replay profiles for the volume.
    Free Disk SpaceIndicates the available free disk space.
    Online (Only For EqualLogic Storage Array)Indicates whether the resource is available online or not.
    Group Members (Only for EqualLogic Storage Array)Indicates the group members added for EqualLogic storage array.
    Free Group Space (Only For EqualLogic Storage Array)Indicates the available group space.
    SP A IP Address (Only VNX storage)Indicates the SP A IP address.
    SP B IP Address (Only VNX storage)Indicates the SP B IP address.
    LUNs (Only for VNX and Unity storage)Indicates the number of LUNs created.
    Management IP (Only for Unity storage)Indicates the Management IP for Unity storage.

  4. In the right pane, click View Details.

    The Storage Details page displays the detailed information about the server in the following tabs:

    Field Description
    Summary

    Displays the following information:

    • Storage Information
    • Volumes (Only for Compellent and EqualLogic Storage Array)
    • LUNs (Only for VNX and Unity)
    • Storage Pool Capacity (Only for VNX and Unity)
    • Storage Pools (Only for VNX and Unity)

    To start Storage Center application, click Launch Storage Center

    VolumesDisplays the following information:
    • Name
    • Logical Size
    • Replay Profile
    • Storage Profile
    • Disk Folder
    LUNs (Only for VNX and Unity)Displays the following information:
    • State
    • LUN Name
    • ID
    • Capacity
    File SystemsDisplays the following information:
    • State
    • Name
    • ID
    • Capacity
    • Allocated
    • Pool
