Discovering the OpenManage Enterprise application

Once VxFlex Manager discovers the OpenManage Enterprise application, you can open the application and manage critical and error alerts for the device.

Before you discover the OpenManage Enterprise application, verify that the application is configured and running.

  1. On the menu bar, click Resources and then click All Resources.

  2. Click Discover. The Discovery wizard opens.

    1. On the Welcome page, read the instructions and click Next.

    2. On the Identify Resources page, click Add Resource Type.

    3. From the Resource Type list, select Element Manager.

    4. In the IP Address Range box, enter the IP address of the OpenManage Enterprise application that you want discovered.

    5. From the Resource State drop-down list, select Managed.

    6. To create a new credential, click the + sign to the right of the Credentials drop-down list.

    7. Click Next > Finish.

    You can check the status of the discovery by going to Settings > Logs. When discovery completes, the OpenManage Enterprise application is listed under Resources.

Refer to the VxRack FLEX Administration Guide for more information about alerts and setting alerting policies in the OpenManage Enterprise application.
