Editing proxy settings

If your network uses a proxy node for external communication, then you must enter the critical information to enable communication with VxFlex Manager virtual appliance.

  1. On the menu bar, click Settings and then click Virtual Appliance Management.

  2. On the Virtual Appliance Management, in the Proxy Settings section, click Edit.

  3. Select Use HTTP Proxy Settings.

  4. In the Node Address (IP or Hostname) box, enter an address for the proxy node.

  5. In the Port box, enter a valid port number from 1â€“65535. Commonly used ports for a proxy node are 80 and 8080.

  6. If the proxy node requires credentials to log in, select Use Proxy Credentials and then in User Name and Password boxes, enter the required user name and password. To verify the password, enter the password in Confirm Password.

  7. To validate the settings entered on this page, click Test Proxy Connection.

  8. Click Save.
